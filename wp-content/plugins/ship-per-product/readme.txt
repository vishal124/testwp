Allows you to set shipping cost to each products based on customers location.

== Description ==

Ship Per Product allows to set shipping cost based on shipping location for each product.
All these features are available on product page.

= Main features =
* **Cost for simple and variable product can be set for each customer location**.
* **Can set default shipping cost to products if not available for any customer location from admin setting**.
* Separate Table for adding shipping rates under the shipping setting tab.
* Can Include/Exclude tax from shipping cost.
* **Calculate handling fee per order and if per order is not set, calculate per product handling fee**.
* **Can restrict shipping to only specific countries**.
* **Can skip shipping cost for free shipping products**.
* Shipping cost for each user location can be imported using csv file using built in csv importer-exporter in admin setting.
* Easy to use and install
* Easy to configure

= Premium Features =
* Can set shipping cost of simple products for each customer location on **Product edit page**.
* Can import/export rate on product edit page.
* Can set shipping cost of each **variation**  for each customer location on **Product edit page**.
* Can import/export rate for each variations.
* **If shipping cost of a variable product is not defined then shipping cost will be it's parent product's shipping cost**.
* Separate Table for adding shipping rates under the shipping setting tab for both product based and category based shipping where all rates are listed.
* Shipping log is mantained.
* Can set shipping cost for each category in separate **Category Based Shipping** Tab.
* Shipping cost will be applicaple for each product under the selected category.
* Can set shipping rate on **bulk products**.

== Installation ==

= Automatic installation =

Automatic installation is the easiest option as WordPress handles the file transfers itself and you don't need to leave your web browser. To do an automatic install of Ship Per Product, log in to your WordPress dashboard, navigate to the Plugins menu and click Add New.

In the search field type **Ship Per Product** and click Search Plugins. Once you've found our Ship Per Product Extension you can view details about it such as the point release, rating and description. Most importantly of course, you can install it by simply clicking "Install Now".

= Manual installation =

The manual installation method involves downloading our Product Auto Share plugin and uploading it to your webserver via your favourite FTP application. The WordPress codex contains [instructions on how to do this here](http://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation).
